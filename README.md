## About Me

I am a third-year Computer Science student at Amrita Vishwa Vidyapeetham with a focus on full-stack web development and applied AI. My work spans building production-grade web applications, designing modern responsive interfaces, and engineering AI agents that integrate into real-world workflows.

I work at the intersection of software engineering and language model systems — building specialized agents that automate tasks and connect into existing products through well-designed APIs. My priority is always shipping things that work, not just prototyping concepts.

**Currently working on:**
- Full-stack web applications with clean, scalable backend architecture
- Modern and responsive UI design systems
- Task-specific AI agents using LangChain and RAG pipelines
- Embedding AI capabilities into production web services
