## About Me

I am a third-year Computer Science student at Amrita Vishwa Vidyapeetham with a focus on full-stack web development and AI systems. My current work spans building production-grade web applications, designing modern responsive interfaces, and developing AI agents that integrate directly into real-world workflows.

I am particularly interested in the intersection of language models and practical software — specifically, building specialized AI agents that can automate tasks and plug into existing products. My goal is to ship things that work, not just prototype concepts.

**Currently working on:**
- Dynamic full-stack web applications with clean backend architecture
- Modern, responsive UI design systems
- Task-specific AI agents using LangChain and related frameworks
- Integrating AI capabilities into production web services via APIs
